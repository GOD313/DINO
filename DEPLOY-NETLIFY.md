# Deploy on Netlify

این ریپو برای Deploy روی Netlify آماده است.

## تنظیمات

- Build command: `npm run build`
- Publish directory: `public`
- Functions directory: `netlify/functions`

## Environment Variables

```env
ADMIN_USER=admin
ADMIN_PASSWORD=ChangeMe-Strong-Password
APP_SECRET=replace-with-a-long-random-secret
SITE_BASE_URL=https://my.karatshod.click
TELEGRAM_RELAY_URL=
```

## مسیرها

- `/` پنل مادر
- `/ports/mehr-hiring/` پورت استخدام مهر
- `/api/status` تست API

## نکته

این نسخه دمو/فنی است و بدون مجوز رسمی نباید به عنوان سامانه رسمی هیچ سازمان یا نهاد معرفی شود.
